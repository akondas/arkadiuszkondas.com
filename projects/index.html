Repman

PHP Repository Manager: packagist proxy and host for private packages

Repman is a PHP repository manager. Main features: free and open source, works as a proxy for packagist.org (speeds up your local builds), hosts your private packages, allows to create individual access tokens, supports private package import from GitHub, GitLab and Bitbucket with one click, REST API and security scanner
